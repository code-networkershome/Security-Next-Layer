# SNL Project Orchestration

This project is divided into two parts:

## 1. Backend (FastAPI + ProjectDiscovery Tools)
Located in `./backend`

### Commands
```bash
cd backend
# Make sure tools are installed (see setup.md)
# Ensure you have your .env file setup with your keys
source ../venv/bin/activate
python3 main.py
```

## 2. Frontend (React + Tailwind + Framer Motion)
Located in `./frontend`

### Commands
```bash
cd frontend
npm install
npm start
```

## Security Warning
This tool is for educational and authorized security testing only. Do not use without permission.
In environments like CTFs or authorized pentests, ensure you follow the rate-limiting rules established.
